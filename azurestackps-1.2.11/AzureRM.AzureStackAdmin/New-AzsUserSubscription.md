# New-AzsUserSubscription

## SYNOPSIS
Creates a user subscription as an administrator.

## SYNTAX

```
New-AzsUserSubscription -Owner <String> -OfferId <String> [-DisplayName <String>] [-SubscriptionId <String>]
 [-DelegatedProviderSubscriptionId <String>] [-DefaultProfile <IAzureContextContainer>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [-PipelineVariable <String>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The New-AzsUserSubscription cmdlet creates a subscription as an administrator.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
New-AzsUserSubscription -Owner "user1@contoso.com" -OfferId $OfferId -DisplayName "Displayname"
```

Description

-----------

The example creates a subscription for the specified user.
